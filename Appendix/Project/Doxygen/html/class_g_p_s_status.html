◆ fix_

double GPSStatus::fix_
private

Describes how good the connection is.

fix can be any of these:
0 = invalid
1 = GPS fix (SPS)
2 = DGPS fix
3 = PPS fix
4 = Real Time Kinematic
5 = Float RTK
6 = estimated (dead reckoning) (2.3 feature)
7 = Manual input mode
8 = Simulation mode

More information can be found in the NMEA GGA telegram http://www.gpsinformation.org/dale/nmea.htm#GGA
