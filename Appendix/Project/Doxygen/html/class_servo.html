Detailed Description

describes a servo motor that is controlled via pwm

A Servo is an implementation of the IPositionMotor interface which describes all motors where you set a position
it is also an implementation of the IMotorStatusGetter interface which describes the GetStatus function used for getting the current status of motor.

See also
IPositionMotor, IMotorStatusGetter
