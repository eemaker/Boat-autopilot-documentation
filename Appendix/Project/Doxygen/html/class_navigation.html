Detailed Description

An implementation of the INavigation interface.

This is an implementation of the INavigation interface.
This module is responsible for receiving data from the GPS, calculating paths, ete's, xte, bearings, setting system parameters,
calling TransmitFromNav in the Transmitter, and calling Run and Stop in the Autopilot.
Because Navigation and JSONTransmitter must call function in each other, JSONTransmitter.h has a forward declaration of INavigation,
Navigation.h has a forward declaration of ITransmitter.h, JSONTransmitter.cpp includes INavigation.h, Navigation.cpp includes
ITransmitter.h. The Navigation calls NavAcquisition in JSONTransmitter with a this-pointer parameter, and JSONTransmitter then
saves this object pointer. Through this method, the two objects hold a reference and pointer to each other, respectively.

See also
JSONTransmitter.h
