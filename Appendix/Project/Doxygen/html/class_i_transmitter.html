Detailed Description

An interface for the Transmitter unit of the system.

This module is responsible for writing data to the fromNav.json file.
Because Navigation and JSONTransmitter must call function in each other, JSONTransmitter.h has a forward declaration of INavigation,
Navigation.h has a forward declaration of ITransmitter.h, JSONTransmitter.cpp includes INavigation.h, Navigation.cpp includes
ITransmitter.h. The Navigation calls NavAcquisition in JSONTransmitter with a this-pointer parameter, and JSONTransmitter then
saves this object pointer. Through this method, the two objects hold a reference and pointer to each other, respectively.

Called by Navigation perform task in all three overloads with a timestamp string.
This functionality has the responsibility of collecting all relevant data from the necessary classes in the system, package
them in accordance with the fromNav.json protocol, and overwrite the json file with the new data.
The classes that information is pulled from are DCMotor, Servo, GPS, and Navigation.
