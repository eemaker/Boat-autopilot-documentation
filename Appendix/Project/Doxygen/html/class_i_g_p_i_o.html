Detailed Description

An interface towards hardware general purpose I/O.

The interface is create to use pigpio, so that we are able to test the functionallity of classes depending on it with out having GPIO hardware.

Note
This interface could be expanded to add more GPIO functionallity, but for this first implementation of CAPTAIN, only pwm and servo were needed
