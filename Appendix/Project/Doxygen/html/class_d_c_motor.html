Detailed Description

describes a DC motor that is controlled via pwm

A DCMotor is an implementation of the ISpeedMotor interface which describes all motors where you set a speed
it is also an implementation of the IMotorStatusGetter interface which describes the GetStatus function used for getting the current status of motor.

See also
ISpeedMotor, IMotorStatusGetter
