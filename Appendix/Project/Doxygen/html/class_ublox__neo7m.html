◆ getGPSData()

void Ublox_neo7m::getGPSData ( )
private

Constantly reads on the serial device and save the date every time there is some.

It only validates NMEA GGA and NMEA VTG telegrams.

See also
http://www.gpsinformation.org/dale/nmea.htm#GGA, http://www.gpsinformation.org/dale/nmea.htm#VTG
